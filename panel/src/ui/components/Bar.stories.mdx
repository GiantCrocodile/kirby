# Bar
### `<k-bar>`

Can be used to create all sorts of toolbars with the option to have three different slots for buttons or other elements: left, center and right.

```html
<k-bar>
  <template slot="left">...</template>
  <template slot="center">...</template>
  <template slot="right">...</template>
</k-bar>
```

All slots are optional.
