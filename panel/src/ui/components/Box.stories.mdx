# Box
### `<k-box>`

The box component is a multi-purpose box with text. You can use it as a foundation for empty state displays or anything else that needs to be displayed in a box.

<Preview>
  <Story name="Regular">
    {{
      template: `
        <k-box>Box text</k-box>
      `
    }}
  </Story>
</Preview>

## Custom content
The box can be filled with your own elements to overwrite the default text implementation

```html
<k-box>
  <!-- custom html goes here -->
</k-box>
```

## Events
The box listens to any typical HTML event.

<Preview>
  <Story name="Events">
    {{
      template: `
        <k-box text="This is a box" @click="$router.push('go/somewhere')" />
      `
    }}
  </Story>
</Preview>
