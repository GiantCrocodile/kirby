# Button Group
### `<k-button-group>`

The Button Group element should always be used when two or more buttons are positioned next to each other. The Button Group takes care of consistent margins between buttons.

<Preview>
  <Story name="Example">
    {{
      methods: {
        edit: action('edit'),
        remove: action('remove')
      },
      template: `
        <k-button-group>
          <k-button icon="edit" @click="edit">Edit</k-button>
          <k-button icon="trash" @click="remove">Remove</k-button>
        </k-button-group>
      `,
    }}
  </Story>
</Preview>
