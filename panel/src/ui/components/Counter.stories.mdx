# Counter
### `<k-counter>`

We use the counter in many fields to show the character count of an input or the accepted min/max length of items.

<Preview>
  <Story name="Regular">
    {{
      template: `
        <k-counter :count="5" :min="2" :max="10" />
      `
    }}
  </Story>
</Preview>

You can use the counter yourself very easily in combination with an input:

```html
<k-counter :count="text.length" :min="2" :max="10" />
<k-text-input v-model="text">
```

<Preview>
  <Story name="With Input">
    {{
      data() {
        return {
          text: "Four"
        }
      },
      computed: {
        styles() {
          return {
            border: "1px solid #ddd",
            background: "#fff",
            padding: ".5rem",
            marginTop: ".5rem"
          };
        }
      },
      template: `
        <div>
          <k-counter :count="text.length" :min="2" :max="10" />
          <k-text-input v-model="text" :style="styles">
        </div>
      `
    }}
  </Story>
</Preview>
