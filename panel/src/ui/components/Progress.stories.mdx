### Setting the value

Instead of passing the current percentage via the `value` prop, you can also use the `set(value)` method:

<Preview>
  <Story name="Set value">
    {{
      template: `
        <div>
          <k-progress ref="progress" :value="50" />
          <k-button-group>
            <k-button @click="$refs.progress.set(0)">0%</k-button>
            <k-button @click="$refs.progress.set(50)">50%</k-button>
            <k-button @click="$refs.progress.set(100)">100%</k-button>
          </k-button-group>
        </div>
      `,
    }}
  </Story>
</Preview>
