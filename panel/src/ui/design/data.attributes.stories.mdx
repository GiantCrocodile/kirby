# Data attributes

Data attributes that you can use as CSS selectors in your custom CSS file.

## Translations

```css
/* Current translation language: English */
.k-panel[data-translation=en] {}

/* Current translation language is default language */
.k-panel[data-translation-default] {}

/* Current translation language is not the default language */
.k-panel:not([data-translation-default]) {}

/* Current translation language is not the default language and this field is non-translatable */
.k-panel:not([data-translation-default]) .k-field:not([data-translate]) {}
```
