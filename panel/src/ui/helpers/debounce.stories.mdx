# debounce
### `this.$helper.debounce`

Debounce function calls with this helper.

#### Parameters

| Name  | Description
| ----  | ---- |
| `fn`  | Function to be debounced |
| `delay` | Delay in miliseconds |

#### Example

```js
this.$helper.debounce(function (query) {
  this.search(query);
}, 200)
```
