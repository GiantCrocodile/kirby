# string
### `this.$helper.string`

Helper that offers three methods:

### `camelToKebab`

```js
return this.$helper.string.camelToKebab("myString);
// result: "my-string"
```

### `isEmoji`

```js
this.$helper.isEmoji(ðŸ™Œ)
// result: true
```

### `lcfirst`

```js
return this.$helper.string.lcfirst("Hello");
// result: "hello"
```

### `template`

```js
return this.$helper.string.template("Walk from {{ a }} to {{ b }}", {
  a: "Berlin",
  b: "Paris"
});
// result: "Walk from Berlin to Paris"
```


### `ucfirst`

```js
return this.$helper.string.ucfirst("hello");
// result: "Hello"
```
