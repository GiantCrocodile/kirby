# upload
### `this.$helper.upload`

Helper to upload a file.

#### Parameters

| Name  | Description
| ----  | ---- |
| `file` | File object |
| `params` | Options object (see defaults below) |

```js
// available params and their default
const defaults = {
  url: "/",
  field: "file",
  method: "POST",
  accept: "text",
  attributes: {},
  complete: function() {},
  error: function() {},
  success: function() {},
  progress: function() {}
};
```
